# Skills

Un skill solo puede crearse si:
1. No existe uno equivalente.
2. El Orquestador lo justifica.
3. El Super Admin lo aprueba.
4. Tiene permisos mínimos.
5. Tiene entradas, salidas y criterio de éxito claros.[11][1]
