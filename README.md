# HCE Agentic Stack

Sistema para operar una Historia Clínica Electrónica con agentes IA especializados, orquestación central y permisos mínimos.

## Componentes
- Orquestador.
- Arquitecto.
- Seguridad.
- FHIR/MCP.
- Producto Clínico.
- UX/HCE.
- Integraciones.
- QA/Test.
- DevOps/Release.

## Base técnica
- FHIR.
- Keycloak / OIDC.
- SMART on FHIR.
- Kubernetes.
- PostgreSQL.
- RabbitMQ o Kafka.
- Mermaid para diagramas.[5][6][7][2]
